Configuração da Aplicação Azure

Regista a app no Azure AD e preenche as credenciais

Como registar a aplicação no Azure:
1. Vai a portal.azure.com → Azure Active Directory → Registos de Aplicações
2. Clica "+ Novo registo" → dá um nome → "Contas neste directório"
3. Em URI de Redireccionamento: seleciona "SPA" e coloca <URL desta página>
4. Copia o Application (client) ID e o Directory (tenant) ID
5. Em API permissions, adiciona: User.ReadWrite.All, Group.ReadWrite.All, Directory.ReadWrite.All, Organization.Read.All → Grant admin consent

AD Bridge Service — SRVH-DC01

Ligação ao serviço local para gerir o Active Directory on-premises

Como instalar:
1. Copia a pasta AD-Bridge para o SRVH-DC01
2. Edita o config.json com as credenciais do AD
3. Corre o 1_INSTALAR.bat como Administrador
4. Preenche abaixo o endereço IP do SRVH-DC01 e clica "Testar Ligação"
User.ReadWrite.All Group.ReadWrite.All Directory.ReadWrite.All Organization.Read.All AdministrativeUnit.ReadWrite.All
